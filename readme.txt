Display top 200 cryptocurrency data with customizable widgets for real-time updates and engaging presentation.

== Description ==

This plugin allows you to display top 200 cryptocurrency data using a variety of customizable widget blocks. These blocks are designed to present real-time cryptocurrency information in an engaging and user-friendly format.

Features:
- **Customizable Widgets:** Tailor each widget block to display different types of cryptocurrency data.
- **Real-Time Data:** Keep your cryptocurrency information up-to-date with real-time data.
- **User-Friendly Interface:** Easily add and configure widgets using the WordPress block editor.
- **Ticker:** Display a scrolling ticker of cryptocurrency prices for a dynamic, real-time update experience.
- **List:** Show a list of selected cryptocurrencies with their current prices and other relevant details.
- **Label Design:** Customize the appearance of labels and titles for a visually appealing and consistent look across your site.

== Installation ==

1. **Upload Plugin:**
   - Download the plugin ZIP file.
   - Go to the WordPress admin dashboard.
   - Navigate to **Plugins > Add New**.
   - Click **Upload Plugin** and select the ZIP file.
   - Click **Install Now**.

2. **Activate Plugin:**
   - After installation, click **Activate Plugin**.

3. **Add Widget Block:**
   - Navigate to any post or page editor.
   - Click the **Add Block** button and search for "Cryptocurrency Widget".
   - Select the desired widget block and configure its settings.

== Usage ==

1. **Adding Widgets:**
   - Go to the WordPress block editor for the page or post where you want to add the widget.
   - Click the **Add Block** button (usually a plus sign) and search for "Cryptocurrency Widget".
   - Choose the widget block you want to add.

2. **Configuring Widgets:**
   - After adding the widget block, configure its settings using the block editor on the right sidebar.
   - Customize options such as widget type, selected cryptocurrencies, and display limits.

3. **Saving Changes:**
   - Once configured, click **Update** or **Publish** to save changes and display the widget on your site.

== Frequently Asked Questions ==

= What kind of cryptocurrency data can I display? =
You can display various types of cryptocurrency data such as ticker symbols, current prices and more. The widget is highly customizable to fit your needs.

= How do I update the plugin? =
To update the plugin, go to **Plugins** in your WordPress admin dashboard. If an update is available, you will see a notification. Click **Update Now** to get the latest version.

= Is the plugin compatible with all themes? =
The plugin is designed to be compatible with most themes. However, if you experience any issues, ensure your theme is updated to the latest version or contact support.

= How frequently is the data updated? =
The data is updated at 10-minute intervals.
